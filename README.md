# Yoodo Website Backup

This repository contains an exact snapshot of the final state of the **yoodo.com.my** website as it existed in **October 2025**.

## 📁 About

This archive includes:

- ✅ The complete **homepage** (`index.html`)
- ✅ The fully functional **plan calculator**
- ✅ All **blog posts** and **published content**
- ✅ Static assets such as images, stylesheets, and JavaScript files

## 📦 Purpose

This backup preserves the final version of the Yoodo website before it was sunset. It serves as a historical reference for users, developers, and researchers interested in the site's design, features, and content.
